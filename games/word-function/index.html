🧠 関数ゲーム（ことばバージョン）

🌟 ゲームの目的

関数カード（概念変換ルール）を使って、指定された「Inputの言葉」を「Outputの言葉」に変換しよう！

変換に成功すれば、使用した関数の枚数が得点となります。最も得点を獲得したプレイヤーが勝利！

👥 プレイ人数

  • 2人以上
  • 各ラウンド、時計回りにプレイヤーが「Input／Output担当」となり、他のプレイヤーが変換ルートを考えます

🔢 関数カード一覧（使用時は番号で記述）

No. 関数カード名 効果例（3つ）
1 f(x) = xの上位概念に変換する ・母 → 人間　・犬 → 動物　・いちご → 果物
2 f(x) = xの下位概念に変換する ・人間 → 母　・動物 → 犬　・果物 → いちご
3 f(x) = xを構成する言葉に変換する ・機械 → ネジ　・学校 → 先生　・カレー → スパイス
4 f(x) = xが属している言葉に変換する ・ネジ → 機械　・先生 → 学校　・スパイス → カレー
5 f(x) = xの同類に変換する ・ゴリラ → 人間　・バナナ → りんご　・電車 → バス
6 f(x) = xと対象的な言葉に変換する ・白 → 黒　・猫 → 犬　・暑い → 寒い
7 f(x) = xから共起（連想）する言葉に変換する ・夏 → 海　・クリスマス → ケーキ　・本 → 図書館

⏪ ゲームの流れ

1. お題の決定（Input／Output）

  • ラウンドごとに時計回りで担当者を交代
  • 現在のプレイヤー：Input（変換元）を決定
  • 右隣のプレイヤー：Output（変換先）を決定
  • 残りのプレイヤーはその変換ルートを考える

2. 解答の記述

各プレイヤーは、自分の紙に以下の形式で記述：

Input：りんご
1: バナナ
3: ゴリラ
2: 哺乳類
Output：哺乳類

※ 数字は関数カードの番号を意味する

3. 同時発表！

  • 全員が「せーの！」で同時に紙を公開して発表
  • 使用関数の順番と変換語を説明

4. 判定・得点

  • 関数の種類制限（連続・繰り返し）のルールを守っているか？
  • 変換が成立していると納得できるか？
  • 使った関数の数を得点として記録

⚠️ 変換ルールの注意事項

✨ 同じ関数カードは連続で使えない

  • ❌ NG：3 → 3
  • ❌ NG：1 → 3 → 1（1が2つおき）
  • ✅ OK：1 → 3 → 2 → 6（すべて異なる）

✨ 同じ言葉を2回以上使うのは禁止

  • ❌ NG：りんご → 赤い → りんご

🏆 勝利条件

複数ラウンド行い、最も多くの得点を獲得したプレイヤーが勝利！

📦 必要なもの

  • 紙とペンだけ！
